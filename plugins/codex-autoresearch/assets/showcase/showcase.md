# Showcase Assets

`dashboard-demo.png` is the public README screenshot. It should show the dashboard as a real operator surface: metric trend, best/latest context, trust state, and recent decisions.

When the dashboard UI changes, refresh the screenshot from a demo or local sample session and keep the frame focused on the useful surface. Avoid screenshots that expose local absolute paths, personal data, browser chrome, or cropped text.

The plugin manifest may use compact SVG or PNG screenshots, but the root README should prefer this full dashboard image so people can understand the product before installing it.
